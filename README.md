### Built With

- React
- Redux Toolkit
- React Router
- TypeScript
- Styled Components
- Firebase
- React Slick

### What will you learn

- How to use Styled Components
- How to setup Redux Toolkit to create a store
- How to authenticate using Firebase
- How to get data from Firestore

### Steps

- Setup Routing
- Login Page
- Header component
- Nav Menu
- Setup Firebase
- Home Page

  - Categories
- Details
